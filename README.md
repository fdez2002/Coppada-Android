<h1>Coppada</h1>
<p><br></p>
<h2>&iquest;Qu&eacute; es?</h2>
<p>Coppada es una aplicaci&oacute;n m&oacute;vil relacionada con la b&uacute;squeda y recomendaci&oacute;n de restaurantes y negocios.</p>
<p><br></p>
<h2>&iquest;C&oacute;mo funciona?</h2>
<p>Utiliza la API de Yelp para obtener informaci&oacute;n sobre los lugares, como restaurantes y negocios, as&iacute; como para buscar y mostrar los resultados en la aplicaci&oacute;n. La API de Yelp proporciona datos detallados sobre los lugares, incluyendo su nombre, direcci&oacute;n, calificaci&oacute;n, rese&ntilde;as, fotograf&iacute;as etc.</p>
<p><br></p>
<h2>&iquest;Qu&eacute; puedes hacer?&nbsp;</h2>
<p>La aplicaci&oacute;n permite a los usuarios acceder a una variedad de funcionalidades, como buscar lugares cercanos, ver rese&ntilde;as y recomendaciones, y a&ntilde;adir a favoritos a los restaurantes o negocios que el usuario desea.</p>
